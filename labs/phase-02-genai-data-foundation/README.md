# Lab Phase 02: GenAI and Data Foundation

## Objective

Add GenAI explanation and RAG while keeping evidence authoritative.

## Tasks

1. Create a Bedrock model access plan.
2. Create prompt templates with required citation instructions.
3. Configure Bedrock Guardrails for PII and unsafe content.
4. Create a Knowledge Base for approved policy documents.
5. Build a model evaluation pack.
6. Record model ID, prompt version, guardrail version, and source documents.

## Acceptance criteria

- Every AI explanation cites source evidence or clearly says evidence is insufficient.
- Guardrail behavior is tested before demo.
- Token usage is visible by workflow.
- Human approval remains required.
