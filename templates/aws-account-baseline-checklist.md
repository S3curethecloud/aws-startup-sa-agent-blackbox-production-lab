# AWS Account Baseline Checklist

- [ ] Root MFA enabled.
- [ ] IAM Identity Center or least-privilege admin roles configured.
- [ ] CloudTrail enabled.
- [ ] GuardDuty enabled.
- [ ] Security Hub enabled.
- [ ] AWS Config evaluated for production accounts.
- [ ] Budgets configured.
- [ ] Cost allocation tags activated.
- [ ] S3 Block Public Access enabled account-wide.
- [ ] KMS key strategy documented.
- [ ] Secrets Manager used for secrets.
- [ ] VPC/network design reviewed.
- [ ] WAF plan created for public endpoints.
- [ ] Backup and retention requirements documented.
- [ ] Incident contacts documented.
