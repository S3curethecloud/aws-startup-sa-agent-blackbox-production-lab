# Lab Phase 00: Consultation

## Objective

Practice the startup SA founder call and produce the first architecture decision set.

## Steps

1. Review `assets/securethecloud_phase2_roadmap.png`.
2. Complete `templates/founder-intake-form.md`.
3. Write the first customer workflow.
4. Complete `templates/prfaq.md`.
5. Record decisions in `docs/10-decision-log.md`.

## Acceptance criteria

- Founder problem is clear.
- MVP scope is less than six weeks.
- Non-scope is explicitly documented.
- Cost ceiling is known.
- Enforcement claims are blocked until later phase.
